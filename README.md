# Talkie Speech module

Audio capture, STT, TTS, speaker filter, and voice calibration. Self-contained module for [talkie-core](https://github.com/talkie-assistant/talkie-core); config and help are merged by discovery.

**Documentation:** See the [project wiki](https://github.com/talkie-assistant/talkie-core/wiki) â€” [Module Speech](https://github.com/talkie-assistant/talkie-core/wiki/Module-Speech), [Module Standard](https://github.com/talkie-assistant/talkie-core/wiki/Module-Standard), [Module API](https://github.com/talkie-assistant/talkie-core/wiki/Module-API).

**Run:** Use from talkie-core (clone as submodule under `modules/speech` or install via Marketplace). Do not copy `modules/api` or `sdk`; the main app provides them.
